# STREAMFLIX

It's our Mini-Project, An Android Streaming application, which shows the same structure &Architecture as NETFLIX. It was built using Technology stack: Kotlin, Jetpack Compose, Supabase. On Android Studio platform supporting AVD API33+. The project is built from scratch with assisst of Github Copilot form it's UI/UX interface to backend Structure.
This project was built in the time period of January'26 ~ March'26 solely by me.
